### “掌上娱乐”APP
```
项目描述：
  为迎合用户在生活中愈加广泛的娱乐需求，实现一个集电影、小说、音乐、图库于一体的移动端APP项目，该项目呢是主要采用vue、HTML5、CSS3、ES6等技术。该项目，可以使用户在平台上浏览到当前最新的电影列表及电影详情；新潮音乐；海量图书；精美图集等等；响应式界面及整体风格色调，使用户体验感大幅度提升；前台主要使用vue全家桶来搭建整体框架，后台通过调用接口来进行数据的导入；

项目功能：
   主要分为四大模块，电影：可浏览到当前最新的电影列表及其详情页，配有图片、简略电影信息；音乐：随时听音乐，悬浮式音乐列表，跃动式歌词，解决了用户随时随地想听音乐的需求，与传统的移动音乐播放器相比，该部分音乐模块有配图、对应歌词等等；小说：当前热播电视剧原剧小说及其小说详情；图库：海量图片集，随时查看精美图片；

技 术 栈：vue全家桶+sass+ES6+CSS3+HTML5
```
